Cascading Style Sheets (CSS)

Homepage

Cascading Style Sheets, beter bekend als CSS, is een stijlbladtaal die wordt gebruikt om de vormgeving en lay-out van webpagina's te beschrijven. Terwijl HTML (HyperText Markup Language) wordt gebruikt om de structuur en inhoud van een document te bepalen, zorgt CSS voor de visuele presentatie, zoals kleuren, lettertypes en de positionering van elementen.

De basisprincipes van CSS

Het belangrijkste doel van CSS is het scheiden van inhoud (HTML) en presentatie. Dit maakt het mogelijk om de lay-out van meerdere pagina's tegelijkertijd te wijzigen door slechts één bestand aan te passen. Dit verbetert de toegankelijkheid, biedt meer flexibiliteit en vermindert de complexiteit van de inhoud.

Hoe ziet een CSS-regel eruit?

Een CSS-regel bestaat uit een selector en een declaratieblok. De selector wijst naar het HTML-element dat je wilt stijlen. Het declaratieblok bevat één of meer declaraties, gescheiden door puntkomma's. Elke declaratie bestaat uit een CSS-eigenschap en een waarde.

Belangrijke CSS-concepten

Om CSS goed te begrijpen, zijn er een aantal kernconcepten die elke webontwikkelaar moet beheersen. Hieronder volgt een opsomming van de meest cruciale onderdelen:

  • Selectors: De manier waarop specifieke HTML-elementen worden gekozen (bijvoorbeeld via class, id of elementnaam).
  • Het Box Model: Het concept dat elk element wordt omringd door padding, borders en marges.
  • Flexbox: Een lay-outmodel ontworpen voor het verdelen van ruimte tussen items in een container.
  • CSS Grid: Een krachtig tweedimensionaal lay-outsysteem voor het web.
  • Responsiveness: Het gebruik van media queries om pagina's aan te passen aan verschillende schermformaten.

Ontwikkeling en Versies

CSS is door de jaren heen sterk geëvolueerd. Wat begon als een eenvoudige manier om tekst op te maken, is uitgegroeid tot een complexe taal waarmee animaties en geavanceerde lay-outs mogelijk zijn.

Versieoverzicht

Tabel 1: De evolutie van CSS-versies door de jaren heen
Versie Jaartal (W3C Rec) Belangrijkste toevoegingen
CSS 1 1996 Basis lettertypes, kleuren en uitlijning.
CSS 2 1998 Positionering (absolute/relative), z-index en media types.
CSS 3 1999 - heden Modules, Flexbox, Grid, Animaties, Schaduwen en meer.

Tegenwoordig wordt er niet meer gesproken over "CSS4" als een enkele versie, maar werkt de W3C (World Wide Web Consortium) aan losse modules die onafhankelijk van elkaar worden geüpdatet.
